RU Direct


Description

RU Direct is an Android app that provides an intuitive and responsive interface for locating information about Rutgers University buses. It displays what times buses are arriving at each bus stop, shows maps of each of the bus routes, and provides directions from one bus stop to another.

Role

I designed the app, built the front-end, programmed the core features, implemented a directions algorithm from scratch, and integrated the third party libraries/APIs.

  • Languages/Frameworks: Java, Android, NextBus API, Google Maps API, JGraphT, OrmLite
  • Date: April 2015 - Present
  • Website: RU Direct
  • Play Store: RU Direct

DJ Crowd


Description

DJ Crowd is a web app that lets people who are at an event decide what music is played. The host of the event specifies a hashtag for the event, and relevant songs are played based on the things that people tweet with the hashtag.

Role

I built the front end and back end of the web app and integrated the Twitter, Musixmatch, and YouTube APIs.

  • Languages/Frameworks: HTML, CSS, JavaScript, Meteor, Linode, Twitter API, Musixmatch API, YouTube API
  • Date: April 2015
  • Website: DJ Crowd

Gtabs


Description

Gtabs adds email tabs to Gmail, which makes it easier for you to switch between emails. It makes you more productive by keeping all of the Gmail emails that you are looking at consolidated in one Chrome tab and letting you reference different emails quickly when you're drafting your own email.

Role

I designed and programmed the extension and integrated the Materialize front-end framework.

  • Languages/APIs: HTML, JavaScript, CSS, Materialize Front-End Framework
  • Date: November-December 2014
  • GitHub: Gtabs
  • Chrome App Store: Download Gtabs

ArtRoom


Description

ArtRoom is an Android app that provides an interactive learning experience for people who visit art museums. Museum curators can attach Estimote Beacons next to each art piece, and when a visitor is near a piece of art, he/she can press a button and get information such as the artist, time period, and curator comments. The purpose of the app is to help people learn about art and to help them become more culturally aware.

Role

I designed the Android app, programmed the front end interface, and integrated the Estimote Beacon API.

  • Languages/APIs: Java, Android, Estimote Beacon API, MongoDB, The Walters Art Museum Collections API
  • Date: November 2014
  • Awards: Best Use of Google API at YHack 2014
  • GitHub: ArtRoom App ArtRoom Server

Friendcast


Description

Friendcast is an Android app that makes it easier to hang out with friends! Whenever you are free, you choose the friends you want to invite and a location to meet at, and it will send a push notification to your friends notifying them that you want to hang out! The goal of the app is to make the process of hanging out with friends easier, more instantaneous, and more spontaneous.

Role

I designed the Android app, programmed the front end interface, and integrated the Facebook API.

  • Languages/APIs: Java, Android, Facebook API, Google Maps API, Parse
  • Date: October 2014
  • Awards: Favorite Yodle Hack - HackRU Fall 2014
  • GitHub: Friendcast
