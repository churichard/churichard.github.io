Gtabs


Description

Gtabs adds email tabs to Gmail, which makes it easier for you to switch between emails. It makes you more productive by keeping all of the Gmail emails that you are looking at consolidated in one Chrome tab and letting you reference different emails quickly when you're drafting your own email.

  • Languages/APIs: HTML, JavaScript, CSS, Materialize Front-End Framework
  • Date: November-December 2014
  • GitHub: Gtabs
  • Chrome App Store: Download Gtabs

ArtRoom


Description

ArtRoom is an Android app that provides an interactive learning experience for people who visit art museums. Museum curators can attach Estimote Beacons next to each art piece, and when a visitor is near a piece of art, he/she can press a button and get information such as the artist, time period, and curator comments. The purpose of the app is to help people learn about art and to help them become more culturally aware.

  • Languages/APIs: Java, Android, Estimote Beacon API, MongoDB, The Walters Art Museum Collections API
  • Date: November 2014
  • Awards: Best Use of Google API at YHack 2014
  • GitHub: ArtRoom App ArtRoom Server

Friendcast


Description

Friendcast is an Android app that makes it easier to hang out with friends! Whenever you are free, you choose the friends you want to invite and a location to meet at, and it will send a push notification to your friends notifying them that you want to hang out! The goal of the app is to make the process of hanging out with friends easier, more instantaneous, and more spontaneous.

  • Languages/APIs: Java, Android, Facebook API, Google Maps API, Parse
  • Date: October 2014 - Present
  • Awards: Favorite Yodle Hack - HackRU Fall 2014
  • GitHub: Friendcast
